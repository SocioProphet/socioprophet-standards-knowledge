# knowledge.v1 JSON-LD context

This directory contains the draft `v1` semantic overlay for the expanded Knowledge Context store surface.

Coverage added in `v1`:
- `ProvenanceRecord`
- `Entity`
- `Passage`
- `Mention`
- `EmbeddingRef`
- `VectorIndexRef`
- `EntityResolutionRecord`

This context extends the `v0` semantic overlay without mutating the frozen `KNOWLEDGE_JSONLD_v0` label.
