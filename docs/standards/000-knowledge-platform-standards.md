# Knowledge Context Standards (v0.1)

This repo specifies:
- Canonical knowledge artifacts: Note, Claim, Annotation, ProvenanceRecord
- Meriotopographic relations (Edge model + predicate registry + constraints)
- Privacy/masking (mask-before-index/embed/export) + human redaction quorum gates
- Tokenization + lexical IR + vector retrieval interfaces
- Office/editor integration as rendering/authoring surfaces (canonical store remains knowledge substrate)

Normative requirements are inherited from the storage standards repo (see 001-upstream-dependencies.md).
